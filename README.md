# Local Places

A web application for discovering and sharing local places.

## Features

- User authentication (login/register)
- Submit new places with images/videos
- Browse and search places
- Admin dashboard for content moderation
- Responsive design
- Secure file uploads
- Password reset functionality
- User profiles

## Requirements

- PHP 7.4 or higher
- MySQL 5.7 or higher
- Web server (Apache/Nginx)
- Composer
